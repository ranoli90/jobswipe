# Fonts

This directory contains the Inter font family files required by the app.

## Required Files

- Inter-Regular.ttf
- Inter-Medium.ttf
- Inter-SemiBold.ttf
- Inter-Bold.ttf

## Installation

Download the Inter font files from [Google Fonts](https://fonts.google.com/specimen/Inter) and place them in this directory.

The app is configured to use these fonts as declared in `pubspec.yaml`.
